# Java SDET Project

This project is a collaborative Java SDET project where team members work together on tasks and improve their problem-solving skills.

## Team Work
- This project is developed as a team.
- Each member will work on their own tasks.
- Everyone should follow clean code and commit regularly.

## Tasks
- Task01: Library system logic
- Task02: Password manager
- Task03: List operations

## Rules
- Each member should create their own branch.
- Do not push directly to main branch.
- Use clear commit messages.
- Complete your task and share updates with the team.

## Technologies
- Java
- IntelliJ IDEA
- Git & GitHub

## Purpose
To improve Java skills, teamwork, and build a strong foundation for SDET.
